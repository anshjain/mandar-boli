PunyaUday fund is establish by Priya Shreeyansh Jain on the day of 50th achrya Shree Vidyasagar Maharaj Diksha divas (i.e. 30th June 2018) at sus gaon pune.
The main motto behind this website is to keep track of funds, which temple is going to receive in future and provide a convenient way for end user to monitor and view the fund status. This will be the convenient way to raise funds for temple and help to avoid negligence of funds.

How it work :


User can search his unpaid donation information by using registered "phone number", it will list down all donation information which are not paid yet. corresponding to registered temples.
Once the list is fetched user can select the checkbox against it and send a message using send link against the row, Please do mention the mode of payment (i.e. Online or Cash) and amount, If you paid online then please do mention transaction ID as well, this will help admin to track you information and avoid discrepancy.
